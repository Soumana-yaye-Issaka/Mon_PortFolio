# Mon Portfolio

Petit site web statique présentant des projets et exemples de travaux.

Ce portfolio est une vitrine digitale haut de gamme conçue pour présenter mon expertise, mes projets et mes services en tant que développeur web fSenior basé à Niamey, Niger.
Contexte & Objectif
L'objectif était de créer une expérience web immersive et mémorable, capable d'impressionner recruteurs et clients dès les premières secondes, tout en reflétant un niveau d'expertise international.

## Description

Ce projet contient une page d'accueil simple en HTML/CSS et une page de portfolio. Il sert de base pour présenter des projets personnels.

## Structure du projet

- `index.html` : page d'accueil.
- `style.css` : styles globaux.
- `Images/` : dossier contenant des images et la page `Portfolio.html`.

## Utilisation

1. Ouvrir `index.html` dans un navigateur web (double-clic ou `File -> Open`).
2. Pour modifier le style, éditer `style.css`.

## Personnalisation

- Remplacer les textes et images par vos contenus.
- Ajouter de nouvelles pages HTML ou sections dans `index.html`.
